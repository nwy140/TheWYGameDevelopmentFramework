Detailed Description

This class contains helper functions about the overall game structure.

For more specific use of individual items such as worlds and levels see the corresponding subfolder.

◆ IsCurrentLevelLastInWorld()

static bool GameFramework.GameStructure.Game.GameHelper.IsCurrentLevelLastInWorld ( )
static

Returns whether this is the last level in the current world based upon the current state.

The game might also be complete if this returns true, so check IsCurrentLevelLastInGame() if you want to distinguish between the two. See also IsNextWorldUnlocked to test whether the next world has already been unlocked before.

If we are not using worlds then this will always return false;
