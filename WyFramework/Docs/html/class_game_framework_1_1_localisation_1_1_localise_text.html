◆ AllowedLanguages

string [] GameFramework.Localisation.LocaliseText.AllowedLanguages
staticget

List of allowed languages.

As the default file can contain additional languages that the game doesn't localise for then we specify what the user can actually choose from. If this is empty then we default to the first language in the user / main localisation file.
