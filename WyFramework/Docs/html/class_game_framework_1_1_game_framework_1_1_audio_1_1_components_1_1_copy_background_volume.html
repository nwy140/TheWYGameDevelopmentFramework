Detailed Description

Copy the global effect volume to an attached Audio Source

The global effect volume is stored on the GameManager component and can be adjusted through the provided in game settings window. This component will copy the users preference and automatically set the attached AudioSources volume to be as specified by the global setting.

Must be attached to a gameobject that contains an AudioSource component.
