Detailed Description

Functionality for getting game feedback from the user.

You can configure any displayed text through the localisation file.

◆ GameFeedbackAssumeTheyLike()

void GameFramework.Social.GameFeedback.GameFeedbackAssumeTheyLike ( )

Should be called when invoked as a direct result of their actions and if you assume the user likes the game.

e.g. a rate button shown only after a certain number of plays.

This displays a popup with just a message and ok button before taking them to the rate page.

◆ GameFeedbackAssumeTheyLikeOptional()

void GameFramework.Social.GameFeedback.GameFeedbackAssumeTheyLikeOptional ( )

Should be called if you assume the user likes the game and when not invoked as a direct result of their actions.

e.g. shown automatically after a set number of plays.

Gives rate, later and remind options.

◆ GameFeedbackUnsureIfTheyLike()

void GameFramework.Social.GameFeedback.GameFeedbackUnsureIfTheyLike ( )

Should be called as a direct result of the user clicking on a rate button when you are uncertain if they like the game.

e.g. a static rate button

Gives a yes/no dialog to check they like. Yes takes them to the rate page, no takes them to a feedback dialog.
