Detailed Description

A class for holding information and working with a collection of localisation entries.

Notes: We could have used a 2d array / matrix to hold values for each language, but would still need a dictionary to reference metadata such as the key (and index into the 2d array) pro's and con's of both, but we just put an entry straight into the dictionary for now.
