Detailed Description

Represents an instance of a dialog.

Allows for animation and managing feedback state.

Dialog Instance should always be on the root of a dialog and be accompanied by an Animator. It should have a child gameobject called Dialog. Check teh existing prefabs for an example of how this should be setup.
